Trainers

Everything about trainers

Schema Definitions

CategoryInfo: object

id: string

category identifier

title: string

category display title

description: string

description of the category

weight: string

ordering weight (ascending)

Example
{
  "id": "GENERAL_MEDAL",
  "title": "Medals",
  "description": "General medals",
  "weight": "30"
}

Errors: object

A collection of errors

errors: Error
Error
Example
{
  "errors": [
    {
      "text": "Invalid stat",
      "field": "stat"
    }
  ]
}

Error: object

Error details

text: string

error text

field: string

field on request to which the error relates

context: object

object containing details of to what the error relates

Example
{
  "text": "Invalid stat",
  "field": "stat"
}

StatInfo: object

id: string

statistic identifier

title: string

statistic display title

description: string

description of the statistic

stat_category: string

category to which the statistic belong

weight: string

ordering weight (ascending)

data_type: string

value type expected

data_unit: string

unit in which the statistic is measured

bronze_threshold: string

for medals only, value required to get bronze

silver_threshold: string

for medals only, value required to get silver

gold_threshold: string

for medals only, value required to get gold

Example
{
  "id": "jogger",
  "title": "Jogger",
  "description": "Walk this far",
  "stat_category": "GENERAL_MEDAL",
  "weight": "10",
  "data_type": "DOUBLE",
  "data_unit": "Km",
  "bronze_threshold": "10",
  "silver_threshold": "100",
  "gold_threshold": "1000"
}

StatValue: object

trainer: string

trainer identifier

stat: string

statistic identifier

timestamp: string (date-tme)

when, in the format YYYY-MM-DD HH:MM and must be in UTC

value: number

Static value

Example
{
  "trainer": "TheBest",
  "stat": "xp",
  "timestamp": "2018-04-20 07:58",
  "value": 1234567
}
